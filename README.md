# AI-Night-Sky-Guide
A location-based constellation guide built using MIT App Inventor

## Overview
This mobile application is developed using MIT App Inventor.
It helps users identify and learn about major constellations
based on their geographical location.

The app is designed as an educational tool for beginners
interested in astronomy and night sky observation.

## Features
- Detects user location using GPS
- Provides location-based sky guidance
- Offers educational descriptions of constellations
- Simple and beginner-friendly UI

## Frontend Design
The frontend is created using MIT App Inventor Designer.
It consists of multiple screens, including Home, Sky Result,
and Guide screens for smooth navigation.

## Backend Logic
- LocationSensor is used to retrieve latitude
- Latitude is passed between screens using start values
- Conditional logic determines hemisphere-based guidance
- Clean and minimal logic is used without overcomplication

## Components Used
- User Interface components (Labels, Buttons)
- Sensors (LocationSensor)
- Control and Logic blocks

## Use Case
The application can be used by students and educators
to assist in learning basic astronomy concepts and
practical night sky observation.

## Future Enhancements
- Adding more constellations
- Seasonal sky visibility
- Map-based location view
